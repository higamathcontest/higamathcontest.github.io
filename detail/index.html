概要

コンテストの目的

本コンテストは、HiGA生を対象に競技数学の楽しさを広めることを目的としています。
競技数学は、授業で学ぶ数学とは異なり、様々な公式や発想力を柔軟に駆使して問題を解く競技です。
本コンテストを通じて、HiGA生が数学の新たな魅力に触れ、数学への興味を深めることを期待しています。

参加対象

本コンテストは、HiGAの全生徒を対象としています。 数学に興味がある生徒はもちろん、これから数学を好きになりたい生徒も大歓迎です。

ルール

出題内容

コンテストでは、競技数学の一般的な下の４つの分野から出題されます。

  • 代数：Algebra
  • 組合せ：Combinations
  • 幾何：Geometry
  • 整数：Number Theory
それぞれの分野において、満遍なく難易度の異なる問題が出題されます。
各分野10問、合計40問の問題が出題されます。
問題を解くのに必要な公式や定理は、事前に配布されている Practice Pack にまとめられています。

コンテストの形式

本コンテストは、オンラインで行われる個人競技です。
各問題には、解答を入力する欄が設けられています。
問題は、コンテスト開始後、指定された時間内に解く必要があります。
コンテスト終了後、解答は自動的に採点され、順位表に反映されます。

得点方式

スコアは、正答した問題の難易度に応じて加算されます。
難易度は、問題の内容を考慮して運営が独自に設定します。
最終的な順位は、獲得したスコアの 合計によって決定されます。
同点の場合は、より早く正答した参加者が上位になります。
しかし、解答の乱れ打ちを防止するため、解答数は 10回 までと制限されます。
また、不正解に対しては、ペナルティとして正答に +5分 時間がかかったとみなされます。

ルール

ルール

HiGA Math Contestのルール・です。
参加する場合は、以下の規約をよくお読み下さい。

第1条. 目的

本規約は、HiGA Math Contest（以下「本コンテスト」）への参加条件および遵守事項を定め、 公平かつ円滑な運営を目的とします。
参加者は、本規約に同意した上で本コンテストに参加するものとします。

第2条. 使用可能ツール

参加者は、コンテスト中に以下のツールのみ使用できます。

  • ツールに記載された電卓・素数表・累乗表
  • 事前配布した Practice Pack

第3条. 禁止事項

参加者は、コンテスト中に以下の行為を行ってはなりません。

3.1 外部との通信

  • 他者との相談・協力
  • SNS・チャット等での情報共有
  • 解答の共有・閲覧

3.2 AI・生成系ツール

以下の使用は禁止します。

  • ChatGPT 等の生成AI
  • 数式自動解答サービス
  • 問題解決を目的としたAIツール全般

例えば、Google検索時に上部に表示される AIによる概要 なども閲覧禁止です。

3.3 不正行為

  • 複数アカウントでの参加
  • なりすまし
  • 問題データの不正取得・共有

3.4 その他

以下の行為は禁止します。

  • 運営への虚偽報告
  • 運営全体への迷惑行為
  • 意図的にサイトを破壊する行為
  • その他、運営が不適切と判断する行為

第4条. 失格・順位取消

以下の場合、運営は参加者を失格または順位取消とすることができます。

  • 本規約への違反が確認された場合
  • 不正行為の疑いが合理的に認められる場合
  • その他、運営が不適切と判断した場合

第5条. 問題・解説の取り扱い

  1. 問題および解説の著作権は運営に帰属します。
  2. 無断での再配布・転載・共有を禁止します。
  3. 学習目的での個人利用は認めます。

第6条. 免責事項

  1. 通信環境や端末不具合による不利益について、運営は責任を負いません。
  2. GitHub Pages 等の外部サービス障害について、運営は責任を負いません。
  3. やむを得ない事情により、コンテスト内容を変更・中止する場合があります。

第7条. 規約の変更

運営は、必要に応じて本規約を変更できるものとします。 重要な変更がある場合は、公式サイト上で告知します。

第8条. 同意

参加登録フォームにおいて「規約に同意する」にチェックした時点で、 本規約のすべてに同意したものとみなします。

プライバシーポリシー

HiGA Math Contest（以下「当コンテスト」）は、参加者の個人情報を適切に保護することを重要視しています。 以下に、当コンテストのプライバシーポリシーを記載します。

1. 収集する情報

当コンテストは、以下の情報を収集することがあります。

  • ユーザー名
  • メールアドレス
  • 提出した解答とその結果
  • コンテストの参加状況

2. 情報の利用目的

収集した情報は、以下の目的で利用されます。

  • コンテストの運営・管理
  • 参加者への連絡
  • コンテストの分析・改善
  • 不正行為の防止

3. 情報の共有

当コンテストは、参加者の情報を第三者と共有することはありません。ただし、法的要請があった場合や、参加者の安全を守るために必要な場合には、情報を開示することがあります。

4. 情報の保護

当コンテストは、参加者の情報を保護するために適切なセキュリティ対策を講じています。ただし、インターネットを介した情報の完全な安全性は保証できないため、参加者は自己責任で情報を提供するものとします。

5. 規約の変更

当コンテストは、必要に応じてプライバシーポリシーを変更することがあります。重要な変更がある場合は、公式サイト上で告知します。
