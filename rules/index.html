ルール

第1条（目的）

本規約は、HiGA Math Contest（以下「本コンテスト」）への参加条件および遵守事項を定め、 公平かつ円滑な運営を目的とします。
参加者は、本規約に同意した上で本コンテストに参加するものとします。

第2条（使用可能ツール）

参加者は、コンテスト中に以下のツールのみ使用できます。

  • 電卓
  • 素数表
  • 累乗表
  • 運営が別途許可した資料

※上記以外の使用可否は「ツール」ページによる記載を優先します。

第3条（禁止事項）

参加者は、コンテスト中に以下の行為を行ってはなりません。

3.1 外 部との通信

  • 他者との相談・協力
  • SNS・チャット等での情報共有
  • 解答の共有・閲覧

3.2 AI・生成系ツール

以下の使用は禁止します。

  • ChatGPT 等の生成AI
  • 数式自動解答サービス
  • 問題解決を目的としたAIツール全般

3.3 不正行為

  • 複数アカウントでの参加
  • なりすまし
  • 問題データの不正取得・共有
  • その他、運営が不正と判断する行為

第4条（失格・順位取消）

以下の場合、運営は参加者を失格または順位取消とすることができます。

  • 本規約への違反が確認された場合
  • 不正行為の疑いが合理的に認められる場合
  • 登録情報に虚偽があった場合
  • その他、運営が不適切と判断した場合

第5条（問題・解説の取り扱い）

  1. 問題および解説の著作権は運営に帰属します。
  2. 無断での再配布・転載・共有を禁止します。
  3. 学習目的での個人利用は認めます。

第6条（免責事項）

  1. 通信環境や端末不具合による不利益について、運営は責任を負いません。
  2. Microsoft Forms や GitHub Pages 等の外部サービス障害について、運営は責任を負いません。
  3. やむを得ない事情により、コンテスト内容を変更・中止する場合があります。

第7条（規約の変更）

運営は、必要に応じて本規約を変更できるものとします。 重要な変更がある場合は、公式サイト上で告知します。

第8条（同意）

参加登録フォームにおいて「規約に同意する」にチェックした時点で、 本規約のすべてに同意したものとみなします。
